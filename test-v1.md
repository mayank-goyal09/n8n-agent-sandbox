Developed a Private-First Retrieval-Augmented Generation (RAG) system that bridges static documentation and generative AI. It utilizes a FAISS-powered vector database for high-speed semantic search and the Mistral-7B/Llama-3 LLM (via Groq API) for hallucination-free responses. The system ensures 100% data sovereignty for querying sensitive internal datasets securely and features a Streamlit interface for an interactive chat experience that converts raw PDFs into an intelligent knowledge base.
